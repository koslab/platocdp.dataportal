platocdp.dataportal Installation
--------------------------------

To install platocdp.dataportal using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``platocdp.dataportal`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        platocdp.dataportal

* Re-run buildout, e.g. with:

    $ ./bin/buildout
